# Netflix Content Analysis

## Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python and Pandas.

## Tools Used

* Python
* Pandas
* Matplotlib
* VS Code
* GitHub

## Dataset Information

* Total records analyzed: 8,807
* Content Types: Movies and TV Shows
* Source: Netflix Titles Dataset

## Analysis Performed

* Dataset exploration
* Missing value analysis
* Content type distribution
* Country analysis
* Rating analysis
* Data visualization

## Key Findings

* Movies significantly outnumber TV Shows on Netflix.
* Some columns contain missing values and require cleaning.
* Netflix content is distributed across multiple countries.
* Content ratings vary across audience groups.
