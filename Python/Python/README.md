# Titanic Survival Analysis

## Project Overview

This project explores the Titanic passenger dataset using Python and Pandas.

## Objectives

- Understand dataset structure
- Identify missing values
- Analyse passenger demographics
- Explore survival patterns

## Tools Used

- Python
- Pandas
- VS Code
- GitHub

## Key Findings

- Analysed passenger information and survival outcomes
- Identified missing data fields
- Calculated average passenger age
- Explored gender distribution

## Skills Demonstrated

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Manipulation with Pandas
- Python Programming
## Results

### Dataset Summary

- Total Passengers: 891
- Average Age: 29.7 years
- Male Passengers: 577
- Female Passengers: 314

### Missing Values

- Age: 177
- Cabin: 687
- Embarked: 2

### Survival Statistics

- Survived: 342
- Did Not Survive: 549
